# 电脑管理工具

在 Windows 上无需安装任何应用、双击即用的电脑管理脚本。通过 LLM 智能诊断"应用无法安装"问题，并辅助用户安全执行修复。

## 快速开始

1. 复制本目录到目标机器
2. 复制 `config.example.json` 为 `config.json`，填入你的 LLM API 信息
3. 双击 `computer_manager.bat`

## 菜单

1. 诊断应用安装问题  （需要配置 LLM）
2. 日常清理维护
3. 软件管理
4. 系统健康快照
5. 查看历史报告
6. 设置
0. 退出
